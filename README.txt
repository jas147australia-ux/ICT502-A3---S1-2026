MEL-B Group 5 - ICT502 Scenario 5 Budget Tracker Application

How to run locally:
1. Unzip the folder.
2. Open index.html in any modern browser such as Chrome, Edge, Firefox or Safari.
3. No server, internet connection, database installation or external library is required.
4. Use the 'Load sample data' button on the Dashboard or About page to demonstrate all functions quickly.

Pages included:
- index.html: dashboard, quick transaction entry, charts, alerts and recent transactions.
- transactions.html: add, edit, delete, search and filter transaction records.
- budgets.html: set monthly and category budgets, then view overspending status.
- reports.html: view monthly report charts/tables and export CSV or JSON.
- about.html: feature summary, technology summary, currency setting and data controls.

Important note:
This is a front-end local-browser prototype prepared according to the user's requirement that the pages should open directly from HTML files in a local browser. Data persistence is implemented through browser localStorage. For a hosted production version, the same interface can be connected to a PHP/MySQL backend.
